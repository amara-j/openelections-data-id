Download Files

We recommend that you right-click on the file link, choose SAVE or SAVE TARGET AS, and save the file to your local PC in a location where you will be able to easily find it. You can then open the file directly by double-clicking on it, or by a right-click and choose OPEN. We have had reports of people experiencing erratic results when opening the larger files (precinct level returns) in a browser window.
